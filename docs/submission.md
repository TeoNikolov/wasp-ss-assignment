# Submission
To submit, send an email to:

- `tnikolov@hotmail.com`
- `tsakovm@gmail.com`

!!! success "Deliverable"
    
    You should submit one `.zip` file containing:
    
    - the `.txt` story script file
    - all `.wav` files
    - all `.bvh` files
    - all `.fbx` files
    
    Refer to the [tutorial](./tutorial.md) for more information on the file naming convention.

!!! abstract "Example"

    ```
    - team_name.zip
        - script.txt
        - bvh/
            - 01A.bvh
            - 02B.bvh
            - 03C.bvh
            - 04B.bvh
        - fbx/
            - 01A.fbx
            - 02B.fbx
            - 03C.fbx
            - 04B.fbx
        - wav/
            - 01A.wav
            - 02B.wav
            - 03C.wav
            - 04B.wav
    ```
